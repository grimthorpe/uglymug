What is cgitelnet?
Is a CGI wich serves as gateway for a telnet session.

Under which license is cgitelnet distributed?

Why is cgitelnet Free Software?
Read this.

Why is cgitelnet useful?
Using cgitelnet, you can do a telnet session in your Web browser. This is useful when you can't do a telnet session from your computer because, example, the telnet program is not installed or your computer is behind a firewall that only allow requests using a Web browser.

What cgitelnet stands for?
CGI telnet session gateway.

Requirements

A Web server cappable to execute CGI scripts and the telnet program.
